## 核心概念

### 区块链基础
- **共识机制**：PoW、PoS、PoA
- **第一层 vs 第二层**：理解扩容方案
- **Gas 优化**：降低交易成本
- **账户抽象**：ERC-4337 及其他方案

### 密码学
- **ECDSA**：椭圆曲线数字签名算法
- **哈希函数**：SHA-256、Keccak-256
- **默克尔树**：数据完整性验证
- **零知识证明**：隐私保护验证

## 跨链开发

### ZetaChain 概览
- **目的**：实现无缝跨链交易
- **架构**：全链智能合约（Omnichain Smart Contracts）
- **核心功能**：
  - 跨链消息传递
  - 流动性聚合
  - 统一状态管理

### 跨链桥接机制
- **流动性桥**：跨链代币交换
- **消息桥**：数据传输
- **包装代币**：在其他链上表示资产

### 常见陷阱
- 滑点与无常损失
- 桥接合约漏洞和安全风险
- 状态同步延迟

## 智能合约

### Solidity 最佳实践
- **重入保护**：防止递归调用攻击
- **整数溢出/下溢**：SafeMath 考虑
- **访问控制**：基于角色的权限管理

### 设计模式
- **工厂模式**：部署多个合约实例
- **代理模式**：可升级合约
- **事件日志**：链下索引

### Gas 优化技术
- 批量操作
- 存储布局优化
- 函数可见性优化

## DeFi 与市场动态

### 流动性池
- **自动做市商（AMM）**：Uniswap、Curve
- **流动性提供者费用**：收入模式
- **滑点**：对交易的影响
- **无常损失**：流动性提供者的风险

### 市场操纵
- **鲸鱼行为**：大额持有者的操作
- **抢跑交易**：交易排序攻击
- **三明治攻击**：操纵技巧
- **MEV（最大可提取价值）**：矿工可提取价值

### 风险管理
- 头寸规模
- 止损设置
- 投资组合多元化
- 项目尽职调查
